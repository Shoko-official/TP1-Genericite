# TP1 - Généricité (Java)

Ce projet a été réalisé dans le cadre d’un **cours de Java**.  

## Contenu du projet
- **Media.java** - Classe de base représentant un média.  
- **Livre.java** - Sous-classe de `Media` pour les livres.  
- **CD.java** - Sous-classe de `Media` pour les CD.  
- **Membre.java** - Représente un membre de la bibliothèque.  
- **Empruntable.java** - Interface définissant le comportement d’un objet empruntable.  
- **MainTP1.java** - Classe principale contenant le point d’entrée du programme.  

## Installation & Exécution
```bash
git clone https://github.com/Shoko-official/TP1-Genericite.git
# Compiler et exécuter la classe MainTP1
````

## Concepts abordés

* Généricité (`<T>`)
* Héritage et polymorphisme
* Interfaces
* Gestion des collections en Java
